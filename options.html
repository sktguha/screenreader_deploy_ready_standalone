SCREENREADER

Seen a great tutorial or ebook online and want to dive in , but finding yourself unable to read after 5 mins??
having eyestrain problems ?? I had the same problem and it really troubled me a lot.
so i created this extension about a year back and have been using myself since , to read(actually hear)
articles , tutorials etc online( No , i am not blind :p)
This is basically what you call a screenreader. Its usage is very simple. To read some text select the text before it with your mouse and then press ' key. The selection automatically jumps and speaks aloud text upto next punctuation mark. 

And after that you can press ' key again to go to next punctuation mark and have the selected text read aloud

Here is a tiny 2 min tutorial(please ignore the mention of an exe in the video. it is not needed)

And here is the keymap, unfortunately it is hardcoded for now

----------KEYMAP-----------
 Keymap is hardcoded for now.
, the line and 'reads it'(i.e upto next full stop)
Similiarly 'm' key goes to previous line and reads it.(i.e upto previous full stop)
' to go upto next punctuation mark
; to go upto previous punctuation mark
shift+i to invert colors
[ key to go to previous word
] key to go to next word
. key to jump back an entire paragraph
/ key to jump forward an entire paragraph .press enter key to stop speech if already speaking. Also if window loses focus then the speech stops automatically

(again please ignore the mention of an exe file in the video. it is not needed)

I have been using it myself for a year now . However it can be of great help even to blind people as you can read from punctuation to punctuation and so it is easier to understand since there is always some logical
break at punctuation mark. Also if you press any other key the selected text will be read again. Again it is useful if you want to hear again and again some , maybe hard to understand , text.
As you see the  main feauture of screenreader is that you need to press only ONE key to do all tasks. So that you can use it just one hand itself and just relax and have the computer read to you.

-------ISSUES---------


If in any page it does not work as expected just click the screenreader icon at top right and you can disable screenreader for the current page and similiarly if you want to enable you can click the screenreader icon again

does not work well at all with safaribooksonline.com unless you deactivate the adding of notes on selecting  text.  It does not work well offline. For offline use(like reading ebooks in html or converted to html from pdf etc), advice you to try the windows specific version mentioned below.
